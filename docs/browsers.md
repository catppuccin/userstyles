# Browsers

> **Note** <br>

Find your browser version by following the instructions below:

<table>
<tr>
 <td> Browser
 <td> Instructions
<tr>
 <td> Chromium
 <td> Type <code>chrome://version/</code>
<tr>
 <td> Firefox
 <td> Type <code>about:support</code>
<tr>
 <td> Opera (GX)
 <td> Type <code>opera://about/</code>
<tr>
 <td> Microsoft Edge
 <td> Type <code>edge://version/</code>
</table>
